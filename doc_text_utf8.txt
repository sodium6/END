📑 เอกสารเปรียบเทียบสิทธิ์การใช้งาน: Admin vs Super Admin
🎯 วัตถุประสงค์
เพื่ออธิบายว่า Admin (ผู้ดูแลระบบ) และ Super Admin มีสิทธิ์เข้าถึงหน้าใดบ้าง ใช้งานฟังก์ชันใดได้เหมือนกัน และส่วนใดเป็นสิทธิ์เฉพาะ Super Admin
✅ ส่วนที่ เหมือนกัน (Admin และ Super Admin ทำได้เท่ากัน)
🔐 จัดการผู้ใช้
สมัครสมาชิก (ยืนยัน/ตรวจสอบ)
จัดการการลืมรหัสผ่าน (รีเซ็ต, ยืนยันการเปลี่ยนรหัสผ่าน)
เข้าสู่ระบบ / ออกจากระบบ
การยกเลิกผู้ใช้งาน (ลบ, ยืนยันการลบ)
📢 ข่าวสารประชาสัมพันธ์
เพิ่มข่าวสาร
แก้ไขข่าวสาร
ลบข่าวสาร
แสดงข่าวสาร
🗂 หมวดหมู่กิจกรรม
สร้างหมวดหมู่กิจกรรม
ยกเลิกหมวดหมู่กิจกรรม
แก้ไขหมวดหมู่กิจกรรม
แสดงหมวดหมู่กิจกรรม
🖼 การจัดการรูปภาพ
เพิ่มรูปภาพ (RESIVE)
ลบรูปภาพ
แสดงรูปภาพ
📑 Portfolio
จัดการ Template Portfolio
📊 รายงาน
สรุปรายงานนักศึกษา
สิทธิ์เข้าถึง: ทั้งสองกลุ่มเห็นและใช้งานได้เหมือนกัน → ถือว่าเป็นสิทธิ์ Admin ทั่วไป
🚫 ส่วนที่ แตกต่างกัน
สิทธิ์เฉพาะ Super Admin
การแจ้งข่าวสารผ่าน Email มหาลัย
Super Admin ✅
Admin ❌
การจัดการผลงานที่ชื่นชอบ (Favorite Works)
ติดดาวผลงานที่ชื่นชอบ
แสดงผลงานที่ชื่นชอบ
Super Admin ✅
Admin ❌
สิทธิ์เข้าถึง: เฉพาะ Super Admin เท่านั้น ไม่ใช่สิทธิ์ Admin ทั่วไป
📝 สรุป
Admin ธรรมดา → ดูแลระบบกลาง: สมาชิก, ข่าวสาร, กิจกรรม, รูปภาพ, Template, รายงาน
Super Admin → มีสิทธิ์เหมือน Admin ทุกอย่าง และเพิ่มสิทธิ์พิเศษ:
ส่งข่าวสารผ่าน Email มหาลัย
จัดการผลงานที่ชื่นชอบ
👉 ดังนั้น Super Admin = Admin + สิทธิ์พิเศษเพิ่มเติม
